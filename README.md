# Set-up Script for Aws/Gcp instances
A Simple Script to setup build environment and install dependencies Files Written in Bash.

## Usage:(for Ubuntu 22.04)
```bash
curl -s https://raw.githubusercontent.com/Adarsh0127-Elite/Setup_script/main/ubuntu22.sh | sudo bash
```

### Usage:(for Ubuntu 24.04)
```bash
curl -s https://raw.githubusercontent.com/Adarsh0127-Elite/Setup_script/main/ubuntu24.sh | sudo bash
```

## Requirements:
- ```curl``` and ```jq``` must be installed.
